Teleport expression evaluators

Small in-browser evaluators for Teleport's predicate languages, built from Teleport's own parser code compiled to WebAssembly. Each runs entirely in your browser.

  • Label expression evaluator

    Evaluate a label expression against a resource's labels, a username, and user traits, the predicate used in Teleport role label matching.

  • App-access rule evaluator

    Evaluate an app-access resource rule against an HTTP request and a caller identity, including path matching with captures.

Source: github.com/juliaogris/teleport-expressions
